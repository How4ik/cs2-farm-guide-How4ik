# Ручной фарм: Avast Sandbox

## Зачем

Способов фарма дропа **два**: ручной и через панели. Ручной — **без дополнительного софта** для автоматизации матчей, но требует **песочницу** — изолированную среду с подменой HWID.

| Способ | Плюсы | Минусы |
|--------|-------|--------|
| **Ручной (Sandbox)** | Нет платы за панель, полный контроль | Долго, много ручной работы, нужна установка Avast |
| **Панели** | Мало участия, AI-сценарии | Стоимость, бан всё равно возможен |

Панели разобраны отдельным этапом: [DM](chapters/26-dm-panel.md) · [Standard](chapters/28-standard-panel.md) · [Bot Forge](chapters/29-bot-forge.md) (наша панель).

**Главная цель песочницы** — основной личный Steam и ферма **не пересекаются**. Один случайный запуск личного клиента в среде фермы может связать то, что вы месяцами разводили.

> ❗ **Важно**
>
> Не смешивайте личную игру и ферму на одном железе **без изоляции**. Основной Steam с друзьями и скинами — **вне** песочницы, на домашнем IP. Ферма — отдельная вселенная.

### Что даёт Avast Sandbox

| Функция | Зачем фермеру |
|---------|---------------|
| Изоляция процессов | Отдельный Steam.exe на аккаунт |
| Подмена / изоляция HWID | Следы не смешиваются между клиентами |
| Несколько песочниц параллельно | До 7 клиентов CS2 (по железу) |
| Бесплатно (с патчем) | Экономия vs панель на малой ферме |

### Ограничения

- Официально продукт **не везде доступен** — нужен патч сообщества
- NVIDIA требует **драйвер 546.65** — иначе вылеты CS2
- Установка — «отдельный квест», нельзя пропустить шаги

## Пошагово

### Этап 1 — Подготовка к установке Avast Sandbox

1. Скачайте **кастомный установщик** Avast (только компонент Sandbox) из проверенного источника / архива сообщества
2. Скачайте **патч** из архива сообщества — инструкция внутри архива
3. Подготовьте папку для бэкапа, если что-то пойдёт не так
4. Закройте все Steam-клиенты и игры

### Этап 2 — Установка (интернет отключён!)

1. **Отключите интернет полностью** (Wi-Fi выкл / кабель вытащен)
2. Запустите кастомный установщик Avast
3. Выберите **только компонент Sandbox** — не тяните лишние модули антивируса
4. Дождитесь окончания установки

> ⚠️ **Внимание**
>
> Лишние модули антивируса потом **мешают**: конфликты с CS2, ложные срабатывания, откат файлов.

### Этап 3 — Патч и самозащита

1. **Примените патч** по инструкции из архива сообщества
2. Откройте настройки Avast
3. **Вырубите самозащиту** — иначе патч и файлы **откатятся** после перезагрузки
4. Перезагрузите ПК при необходимости

### Этап 4 — Критический трюк с config.def

Без этого шага установщик пишет **«продукт не поддерживается в вашем регионе»** и отваливается на полпути.

1. Перейдите в папку установки Avast → подпапка **`Setup`**
2. **Удалите** файл `config.def`
3. **Создайте на его месте пустую папку** с тем же именем: `config.def`
4. Убедитесь: это **папка**, не файл

| Было | Стало |
|------|-------|
| `Setup/config.def` (файл) | `Setup/config.def/` (пустая папка) |

### Этап 5 — Завершение установки

1. **Верните интернет**
2. Запустите Avast Sandbox — проверьте, что модуль активен
3. Создайте **тестовую песочницу** с блокнотом — убедитесь, что изоляция работает
4. Если ошибка региона — проверьте папку `config.def` ещё раз

### Этап 6 — Драйвер NVIDIA (обязательно!)

На видеокартах **NVIDIA**:

| Параметр | Значение |
|----------|----------|
| Версия драйвера | **546.65** (точно) |
| Более свежие сборки | Sandbox часто **вылетает** при старте CS2 |
| Симптом | Думаете, виноват Steam — проблема в драйвере |

1. Удалите текущий драйвер (DDU при необходимости)
2. Установите **546.65** с сайта NVIDIA / архива
3. Отключите автообновление драйвера в Windows
4. Перезагрузите ПК

**AMD** — обычно проще, но тестируйте CS2 в песочнице перед масштабом.

### Этап 7 — Настройка Steam в песочнице

Для **каждого** фермерского аккаунта:

1. Найдите `Steam.exe` → свойства → **«Запуск от имени администратора»**
2. Правый клик по ярлыку → **«Запустить в песочнице Avast»** (или «Run in Sandbox»)
3. Внутри песочницы войдите в **отдельный** Steam-аккаунт
4. Запустите CS2 на **минимальных настройках**:

| Настройка | Значение |
|-----------|----------|
| Разрешение | 800×600 или 1024×768 |
| Качество | Минимум |
| Тени / эффекты | Выкл |
| FPS лимит | 30 (опционально) |

5. **Основной личный Steam** при этом **не трогайте** — он живёт вне песочницы

### Этап 8 — Запуск фарма (7 клиентов)

1. Создайте **7 отдельных песочниц** — по одной на `steam.exe`
2. **Не запускайте** два `steam.exe` в **одной** песочнице
3. Настройте SRT / ночной слот (глава SRT)
4. Режим **2v2** для дропа
5. Следите за температурой CPU — троттлинг удлиняет матчи

### Минимальные требования железа (7 клиентов)

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| RAM | 16 GB | 32 GB |
| CPU | 8 потоков | 12+ потоков |
| VRAM | 6 GB | 8 GB |
| Диск | SSD обязателен | NVMe |

## Чек-лист

### Установка Avast Sandbox

- [ ] Интернет отключён на время установки
- [ ] Установлен **только** компонент Sandbox
- [ ] Патч применён по инструкции
- [ ] Самозащита Avast **выключена**
- [ ] `config.def` — **пустая папка**, не файл
- [ ] Интернет возвращён, Sandbox запускается

### Драйвер и система

- [ ] NVIDIA **546.65** (или AMD протестирован)
- [ ] Автообновление драйвера отключено
- [ ] SSD для shader cache
- [ ] Температуры CPU/GPU в норме

### Изоляция и запуск

- [ ] Каждый аккаунт — **отдельная песочница**
- [ ] Steam.exe — запуск от администратора
- [ ] CS2 на минималках
- [ ] Личный Steam **вне** песочницы
- [ ] Два steam.exe **не** в одной песочнице

### Перед фармом

- [ ] Prime куплен, lvl 2+, обучение снято
- [ ] Пауза 2+ часа после Retake соблюдена
- [ ] SRT / ночной слот настроен
- [ ] Режим 2v2 выбран

## Ошибки

| Ошибка | Последствие | Что делать |
|--------|-------------|------------|
| Пропуск папки `config.def` | «Не поддерживается в регионе» | Удалить файл, создать пустую папку |
| Самозащита Avast включена | Патч откатывается | Вырубить самозащиту |
| Установка с интернетом | Неполная / заблокированная | Интернет off на время install |
| Драйвер NVIDIA новее 546.65 | CS2 вылетает в Sandbox | Откат на 546.65 |
| Два Steam в одной песочнице | HWID и файлы смешаны | 1 песочница = 1 аккаунт |
| Личный Steam в песочнице фермы | Связка аккаунтов | Разделить навсегда |
| 7 клиентов на 8 GB RAM | Вылеты, нестабильный фарм | 16–32 GB или меньше пачка |
| HDD вместо SSD | Shader cache убивает терпение | Только SSD |
| Игнор температуры CPU | Троттлинг, длинные матчи | Мониторинг, меньше клиентов |
| Полный антивирус Avast вместо Sandbox | Конфликты, баны, лаги | Только компонент Sandbox |
| Фарм днём в пик 5v5 | Репорты, длинные матчи | 2v2, ночной слот (SRT) |

> ❗ **Важно**
>
> Avast без папки `config.def` в Setup **просто не установится** — не пропускайте шаг. Это самая частая причина «не работает у меня».

> ⚠️ **Внимание**
>
> Sandbox — **костыль**, но рабочий. На масштабе 50+ аккаунтов часто дешевле панель + железо, чем борьба с вылетами вручную.
