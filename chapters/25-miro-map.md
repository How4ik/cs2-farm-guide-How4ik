## Что на доске

Доска **«Таблица-Гайд Ферма CS2»** — визуальная карта всего процесса фермы в одном месте:

| Блок | Содержание |
|------|-----------|
| **Подготовка** | Прокси, почта, антидетект |
| **Регистрация** | Outlook → Steam → Guard |
| **Пополнение** | Gift, Prime, маркет |
| **Фарм** | Панели, режимы, серверы |
| **Вывод** | Буфер → хранилка → ключи / арбитраж |

Если встроенная схема не загружается — нажмите ссылку выше и откройте доску в Miro.

## Текстовая версия (дубль на сайте)

Ниже — та же логика в виде таблиц, чтобы можно было работать без Miro.

### Полный маршрут аккаунта

| # | Этап | Инструмент | Результат | Время |
|---|------|-----------|-----------|-------|
| 1 | Прокси + регион | Happ Proxy / Node Proxy | IP под Индию | 15 мин |
| 2 | Почта Outlook | Chrome guest | Alias готов | 20 мин |
| 3 | Steam | AdsPower | Аккаунт + оформление | 30 мин |
| 4 | Steam Guard | NebulaAuth | maFile + R-код | 20 мин |
| 5 | Пополнение $5 | Gift card | Маркет открыт (или hold 2 нед.) | 10 мин |
| 6 | Прогрев (опц.) | ASF / Standard Panel | Часы, карточки, lvl | 2–3 нед. |
| 7 | Prime | Steam магазин | Prime Status | 5 мин |
| 8 | 2-й уровень | Retake / DM | Дроп доступен | 40 мин / пачка |
| 9 | Фарм | Sandbox / панель | 1 кейс / неделю | Постоянно |
| 10 | Сбор дропа | Буфер | Транзит безопасен | 1 раз / мес |
| 11 | Вывод | TF2-ключи / TradeON | Деньги в реале | Часы – недели |

### Схема движения дропа

```
Ферма (Prime, фарм)
    ↓  еженедельный дроп
Буфер (без Prime, транзит)
    ↓  раз в месяц, мелкими партиями
Хранилка (тихий аккаунт)
    ↓
Вывод: ключи TF2 (быстро, -40%)  или  TradeON (дольше, -10%…+10%)
```

### Таблица учёта (копия для сайта)

Подробный шаблон — в разделе [Операционка и таблица аккаунтов](chapters/23-operations.md).

Минимум колонок, без которых схема разваливается:

| Колонка | Зачем |
|---------|-------|
| Логин Steam | ID аккаунта |
| Почта / alias | Восстановление |
| Прокси | Не смешать IP |
| R-код Guard | Восстановить 2FA |
| Дата Prime | Окупаемость |
| Буфер | Куда слать дроп |
| Статус | фарм / hold / бан |

### Юнит-экономика на 1 аккаунт

| Статья | Сумма |
|--------|-------|
| Prime (Индия, gift) | ~$15 |
| Prime (арбитраж TradeON) | ~$11.5 |
| Прокси (1 мес) | ~$1–3 |
| Прогрев (опц.) | ~$1–2 |
| Дроп / неделю | 1 кейс |
| Вывод (среднее) | -20% от Steam-цены |

### Связки панелей (схема)

| Масштаб | Рекомендация |
|---------|-------------|
| 3–10 аккаунтов | Ручной фарм (Sandbox) |
| 10–50 | DM Panel |
| 30–100 | **Bot Forge** (наша панель) или Standard + DM |
| 100+ | Bot Forge + буферы по схеме |
