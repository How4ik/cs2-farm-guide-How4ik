# Покупка Prime

## Зачем

**Prime Status Upgrade** — обязательное условие для получения **еженедельного дропа** в CS2. Без Prime аккаунт не участвует в системе наград: кейсы, скины из пула дропа — всё недоступно.

| Без Prime | С Prime |
|-----------|---------|
| Нет еженедельного дропа | 1 кейс / неделю при выполнении условий |
| Деньги заморожены без отдачи | Аккаунт начинает окупаться |
| Фарм бессмысленен | Можно переходить к lvl 2 и фарму |

На индийских аккаунтах Prime **дешевле** российского региона примерно на **$0.92** — на сотне профилей это ощутимая сумма. С арбитражным пополнением через TradeON Prime выходит **~$11.5** вместо **~$15** при gift-картах напрямую.

> ❗ **Важно**
>
> Если Prime куплен, а **уровень Steam ещё первый** — дропа не будет. Вы заморозите деньги на аккаунте без отдачи. Prime покупают **после** подготовки баланса, **перед** прокачкой 2-го уровня и снятием обучения CS2.

### Стоимость Prime: сравнение путей

| Способ пополнения | Стоимость Prime | Время |
|-------------------|-----------------|-------|
| Gift card напрямую | ~$15 | Сразу после активации карты |
| Арбитраж через TradeON (30–40%) | ~$11.5 | +2–3 недели (маркет + hold) |
| Готовый аккаунт с Prime | Дороже всего | 0 — уже куплен |

### Что нужно на кошельке

| Статья | Сумма (инд. аккаунт) |
|--------|---------------------|
| Prime Status Upgrade | ~₹1300 (~$14–15 по курсу) |
| Запас на колебание курса | +₹20–50 |
| После арбитража | Убедиться, что **хватает с запасом** |

## Пошагово

### Шаг 1 — Проверка готовности

1. Откройте таблицу аккаунта — убедитесь:
   - Баланс достаточен (Prime + запас)
   - Маркет открыт (или вы не планируете продавать до открытия)
   - Guard привязан (NebulaAuth, тот же IP)
   - Прогрев завершён или пропущен осознанно
2. Запишите текущий баланс в рупиях

### Шаг 2 — Вход в CS2 с правильной сессией

1. Запустите **AdsPower-профиль** с **тем же прокси**, что при регистрации Steam
2. Проверьте IP на сервисе проверки — страна совпадает с регионом аккаунта
3. Запустите Steam-клиент из этого профиля (или через изолированную среду)
4. Откройте **CS2**
5. **Не выходите** из сессии и **не меняйте IP** до завершения покупки

### Шаг 3 — Открытие витрины Prime

1. В главном меню CS2 найдите баннер **Prime Status Upgrade**
2. Или: Steam → CS2 → страница игры → **Prime Status Upgrade**
3. Проверьте сумму в **индийских рупиях** — должна быть заметно ниже RU-региона
4. Сверьте с таблицей: хватает ли баланса с запасом

### Шаг 4 — Покупка

1. Нажмите **Купить** / **Purchase**
2. Подтвердите списание с кошелька Steam
3. Дождитесь уведомления об успешной покупке
4. Убедитесь, что в профиле CS2 отображается статус **Prime**
5. **Не закрывайте клиент** резко — дайте синхронизации завершиться

### Шаг 5 — Сразу после покупки (не Competitive!)

1. **Не заходите** в Competitive — рано, уровень 1, обучение не снято
2. Переходите к главе **2-й уровень**:
   - Скип обучения CS2
   - Retake или DM для lvl 2
3. Запишите в таблицу:

| Поле | Пример |
|------|--------|
| Дата покупки Prime | 2026-03-15 |
| Сумма в рупиях | ₹1320 |
| Способ пополнения | TradeON / gift |
| Статус | Prime ✅, lvl 1 → нужен lvl 2 |

### Шаг 6 — Масштаб: покупка на пачке аккаунтов

При 10–20+ аккаунтах:

| Правило | Зачем |
|---------|-------|
| Не покупать всем в **один час** | Одинаковый платёжный / активный след |
| Не с одной gift-карты поставщика на 20 профилей | Связка по источнику пополнения |
| Разнести на **2–3 дня** минимум | Естественный паттерн |
| Пачками по 5–7 | Контролируемый процесс |

## Чек-лист

### Перед покупкой

- [ ] Баланс: Prime + запас ₹20–50
- [ ] AdsPower-профиль с тем же прокси, что при регистрации
- [ ] IP проверен — страна совпадает с регионом аккаунта
- [ ] Guard (NebulaAuth) работает, maFile сохранён
- [ ] Прогрев завершён или осознанно пропущен
- [ ] Понимаете: после Prime нужен **lvl 2**, не фарм

### Во время покупки

- [ ] Витрина Prime открыта из CS2 / Steam
- [ ] Цена в **индийских рупиях** адекватна
- [ ] Покупка без смены IP / без выхода из сессии
- [ ] Статус Prime отображается в профиле

### После покупки

- [ ] Дата покупки записана в таблицу
- [ ] Сумма и способ пополнения зафиксированы
- [ ] **Не зашли** в Competitive
- [ ] Запланированы: скип обучения → Retake → lvl 2
- [ ] На пачке: покупки разнесены по дням

## Ошибки

| Ошибка | Последствие | Что делать |
|--------|-------------|------------|
| Prime при lvl 1 без плана прокачки | Дропа нет, деньги заморожены | Сразу Retake / DM после покупки |
| Покупка с другого IP | Подтверждения, урезание трейдов | Только AdsPower + тот же прокси |
| Нет запаса на курсе | Покупка не проходит в пиковый день | +₹20–50 на кошельке |
| Competitive сразу после Prime | Репорты, обучение не снято | Сначала обучение и lvl 2 |
| Prime всем 20 аккаунтам за 1 час | Подозрительный паттерн | 2–3 дня, пачками |
| Одна gift-карта на всю пачку | Связка по пополнению | Разные продавцы / время |
| Покупка без проверки валюты | Переплата на неверном регионе | Рупии, не рубли |
| Забыли записать дату Prime | Не понять, когда отбил вход | Таблица сразу после покупки |
| Prime на буферном аккаунте | Лишняя активность на транзите | Prime только на ферме |
| Ожидание дропа на следующий день | Дроп еженедельный при условиях | Сначала lvl 2 + фарм-условия |

> ⚠️ **Внимание**
>
> Prime на индийском аккаунте с арбитражным пополнением выходит ~$11.5 **только если** вы дождались маркета и не слили hold на неликвиде. Иначе экономия на бумаге.

> ❗ **Важно**
>
> Один аккаунт — **один кейс в неделю** при соблюдении условий дропа. Не стройте план на «окупим Prime за один редкий нож».
