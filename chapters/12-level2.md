# 2-й уровень и снятие обучения CS2

## Зачем

Prime куплен — но до фарма дропа нужны **два обязательных условия**:

| Условие | Зачем |
|---------|-------|
| **2-й уровень Steam** | Valve не выдаёт дроп аккаунтам с lvl 1 |
| **Снятие обучения CS2** | Без этого ограничен доступ к режимам и подозрительна активность |

Без выполнения обоих пунктов Prime **заморожен**: деньги потрачены, дропа нет, фарм не стартует.

### Почему Retake — основной метод

| Метод | Время на аккаунт | Безопасность | Рекомендация |
|-------|------------------|--------------|--------------|
| **Retake** | ~1 раунд (~5–7 мин) | Компромисс | ✅ Основной |
| **Deathmatch** | 3 полных матча | Надёжнее | Долго — часы |
| **Гонка вооружений** | Быстро | Valve видит массовые паттерны | ❌ Не рекомендуется |
| **FunPay ($1)** | Делегирование | Часто читы | ⚠️ Только с записью экрана |

Retake — **лучший компромисс** между скоростью и относительной безопасностью, если вы умеете держать несколько клиентов в изоляции.

### Схема Retake в цифрах

| Параметр | Значение |
|----------|----------|
| Аккаунтов одновременно | **7** (макс.; на слабом ПК — **5–6**) |
| Раундов на аккаунт | **1** |
| Время на всю пачку | ~**40 минут** |
| AFK между переключениями | **Не больше 1 минуты** |
| Пауза после lvl 2 | **2+ часа** перед первым фарм-матчем |

## Пошагово

### Часть A — Снятие обучения CS2

Обучение можно **честно скипнуть** без прохождения всех этапов:

1. Запустите CS2 на аккаунте
2. Дождитесь предложения пройти обучение (или зайдите в него)
3. Откройте **консоль** или меню серверов
4. Зайдите на **любой community-сервер** (не обязательно играть)
5. **Выйдите** с сервера / закройте подключение
6. Игра засчитывает **прохождение обучения**

Проверка: в меню режимов обучение больше не блокирует доступ.

> ℹ️ **Информация**
>
> Скип через «зашёл — вышел» — легитимный способ, который используют и обычные игроки. Не нужны сторонние читы.

### Часть B — Подготовка к Retake (7 аккаунтов)

1. Подготовьте **7 изолированных сред**:
   - Avast Sandbox (отдельная песочница на клиент)
   - Или VM / отдельные инстансы
2. Каждый клиент — **свой Steam**, свой логин, **свой прокси** (если через AdsPower — отдельные профили)
3. Выберите **один регион поиска** для всех 7 — матч найдётся быстрее
4. Убедитесь, что железо тянет 7 клиентов на минималках (см. главу «Железо»)

### Часть C — Алгоритм Retake на пачке

1. Запустите **первый клиент** → CS2 → Поиск матча → **Retake**
2. Дождитесь сервера, **сыграйте один раунд** (постреляли — раунд завершился)
3. **Сверните** клиент (не закрывайте резко — дайте засчитаться)
4. Переключитесь на **второй клиент** — повторите
5. Так по кругу для всех **7 аккаунтов**
6. Между переключениями **не стойте AFK больше минуты** — кик прилетает быстро, матч не засчитается
7. После 7 раундов проверьте уровень Steam на каждом аккаунте

### Часть D — Проверка 2-го уровня

1. Откройте профиль Steam каждого аккаунта
2. Уровень должен быть **2+**
3. Запишите в таблицу: дата, метод (Retake), время на пачку
4. Сделайте **паузу 2+ часа** перед первым фарм-матчем

### Альтернатива — Deathmatch

Если Retake недоступен или нестабилен:

1. Зайдите в **Deathmatch**
2. Сыграйте **3 полноценных матча** на аккаунт
3. Дольше по времени, но засчитывается надёжнее
4. На пачке из 7 — закладывайте **часы**, не минуты

### Делегирование — FunPay (осторожно)

| Шаг | Действие |
|-----|----------|
| 1 | Найдите продавца с хорошими отзывами |
| 2 | **Требуйте запись экрана** процесса |
| 3 | Запретите использование читов в условиях заказа |
| 4 | Проверьте lvl после выполнения |
| 5 | Подождите 2+ часа перед фармом |

> ⚠️ **Внимание**
>
> Слишком часто на FunPay крутят **с читами** — аккаунт умирает через неделю вместе с Prime. Запись экрана обязательна.

## Чек-лист

### Обучение CS2

- [ ] CS2 запущен на аккаунте
- [ ] Зашли на любой сервер и вышли
- [ ] Обучение засчитано — режимы разблокированы

### Подготовка Retake

- [ ] 7 (или 5–6 на слабом ПК) изолированных клиентов
- [ ] Один регион поиска на всю пачку
- [ ] Минимальные настройки графики в CS2
- [ ] Прокси / IP не смешаны между аккаунтами

### Прокачка

- [ ] 1 раунд Retake на аккаунт
- [ ] AFK между переключениями меньше 1 минуты
- [ ] Вся пачка уложилась в ~40 мин (или записана реальная)
- [ ] Уровень Steam **2+** на каждом аккаунте

### После прокачки

- [ ] Дата и метод записаны в таблицу
- [ ] Пауза **2+ часа** перед фармом
- [ ] Не заходили в Гонку вооружений массово
- [ ] FunPay (если использовали) — есть запись экрана

## Ошибки

| Ошибка | Последствие | Что делать |
|--------|-------------|------------|
| 8 клиентов на слабом ПК | Вылеты, кики, раунды не засчитаны | Резать до 5–6 |
| AFK больше 1 мин между аккаунтами | Кик, lvl не поднялся | Быстрое переключение |
| Разные регионы поиска | Долгая очередь | Один регион на пачку |
| Два steam.exe в одной песочнице | HWID перемешан | Отдельная песочница на клиент |
| Гонка вооружений на 20 аккаунтов | Массовый паттерн → бан-волна | Избегать |
| FunPay без записи экрана | Читы → бан через неделю | Требовать видео |
| Фарм сразу после lvl 2 | Резкий скачок активности | Пауза 2+ часа |
| Prime есть, lvl 1 | Дропа не будет | Retake / DM обязательны |
| Пропуск скипа обучения | Блок режимов, странное поведение | Зашёл-вышел с сервера |
| Retake без изоляции от личного Steam | Связка с основным аккаунтом | Песочница / отдельное железо |

> ⚠️ **Внимание**
>
> Retake на слабом ПК с 8 клиентами даст вылеты и кики — режьте до **6**. Лучше две пачки по 5, чем одна на 8 с багами.

> ❗ **Важно**
>
> Режим «Гонка вооружений» для прокачки **сейчас не рекомендуется** — после волн банов Valve слишком хорошо видит массовые одинаковые паттерны.
