# Типичные ошибки

Ошибки, которые встречаются на **каждом** этапе фермы. Одна и та же дыра на 3 аккаунтах — копейки. На 30 — уже больно. На 200 — катастрофа.

---

## Зачем этот раздел

Форумы дают советы кусками. Здесь — связка ошибок по этапам с **последствием** и **решением**. Пройдите чек-лист перед масштабированием.

---

## Регистрация и почта

### Outlook умирает после смены континента

| | |
|---|---|
| **Ошибка** | Сразу после регистрации логинитесь с другого IP / континента |
| **Последствие** | Ящик блокируется, Steam теряет почту |
| **Решение** | Дайте ящику «отлежаться» **2–4 часа** на том же IP |

### Шаблонные имена

| | |
|---|---|
| **Ошибка** | `FarmAcc01`, `CS2Bot_3`, `DropFarmer99` |
| **Последствие** | Бан **цепочкой** по паттерну логина |
| **Решение** | Случайные словосочетания из генератора |

### Бесплатный VPN

| | |
|---|---|
| **Ошибка** | Регистрация на бесплатном VPN с общим IP |
| **Последствие** | Все аккаунты связаны одним следом |
| **Решение** | Платный выделенный IP / прокси, **1 IP = 1 аккаунт** |

### TempMail и одноразовые ящики

| | |
|---|---|
| **Ошибка** | TempMail, TrashMail, FirstMail |
| **Последствие** | Valve режет домены, аккаунт без почты |
| **Решение** | Gmail или Outlook с alias |

### Капча «Что-то пошло не так»

| | |
|---|---|
| **Ошибка** | Долбить одну связку IP + Outlook 10 раз подряд |
| **Последствие** | IP в бане у Microsoft, потерянное время |
| **Решение** | Сменить прокси, начать **заново**, не повторять |

## AdsPower и Steam

### Профиль не закрыт перед сменой прокси

| | |
|---|---|
| **Ошибка** | Переключили прокси, пока AdsPower открыт |
| **Последствие** | Сломанная привязка, смешение следов |
| **Решение** | **Полный выход** из Steam → закрыть окно AdsPower → только потом новый прокси |

### WebRTC не в режиме «Подмена»

| | |
|---|---|
| **Ошибка** | WebRTC отключён или в режиме «Реальный» |
| **Последствие** | Локальный IP просачивается мимо прокси |
| **Решение** | WebRTC → **«Подмена»** в каждом профиле |

### Два профиля на одном IP

| | |
|---|---|
| **Ошибка** | «На минутку» открыли второй профиль на том же IP |
| **Последствие** | Valve связывает аккаунты |
| **Решение** | Строго **1 IP = 1 аккаунт**, без исключений |

### Timezone и язык не под регион

| | |
|---|---|
| **Ошибка** | Индийский прокси, но timezone = Moscow |
| **Последствие** | Несостыковка отпечатка браузера |
| **Решение** | Timezone, язык, валюта магазина = регион прокси |

### Один User-Agent на 20 профилей

| | |
|---|---|
| **Ошибка** | Скопировали один UA на все профили |
| **Последствие** | Одинаковый отпечаток на разных IP — подозрительно |
| **Решение** | Рандомизация UA в AdsPower |

## NebulaAuth и Steam Guard

### Guard с домашнего Wi-Fi

| | |
|---|---|
| **Ошибка** | «Просто быстрее» привязали Guard с домашнего IP |
| **Последствие** | Урезанные трейды, подтверждения по почте |
| **Решение** | Guard **только** с того же IP, что регистрация Steam |

### Письмо Guard открыто без прокси

| | |
|---|---|
| **Ошибка** | Письмо подтверждения в обычном Chrome |
| **Последствие** | Steam запомнил новый след |
| **Решение** | Открывать из **AdsPower** или с IP NebulaAuth |

### Нет бэкапа maFile

| | |
|---|---|
| **Ошибка** | maFile только в NebulaAuth, без копии |
| **Последствие** | Слет Guard → потеря трейдов на аккаунте |
| **Решение** | Бэкап maFile + R-код **после каждой** привязки |

### R-код не сохранён

| | |
|---|---|
| **Ошибка** | R-код показали один раз и забыли |
| **Последствие** | Невозможно восстановить Guard |
| **Решение** | R-код в таблицу + отдельный файл на диске |

### Ферма и хранилка в одной группе NebulaAuth

| | |
|---|---|
| **Ошибка** | Все аккаунты в одной куче |
| **Последствие** | Случайный массовый трейд не туда |
| **Решение** | Группы: `farm_batch_1`, `buffers_march` — **отдельно** |

## Пополнение и gift card

### Карта не того региона

| | |
|---|---|
| **Ошибка** | Турецкая/русская карта на индийском Steam |
| **Последствие** | Блокировка активации |
| **Решение** | Карта **строго** под регион аккаунта |

### Серые магазины без чека

| | |
|---|---|
| **Ошибка** | Купили код без скрина/чека |
| **Последствие** | Код использован → не оспорить |
| **Решение** | Проверенные реселлеры + **скрин до активации** |

### Все карты у одного продавца в один день

| | |
|---|---|
| **Ошибка** | 50 карт одному реселлеру утром |
| **Последствие** | Одинаковый платёжный след |
| **Решение** | Разнести по **времени** и **площадкам** |

### 10 попыток невалидного кода

| | |
|---|---|
| **Ошибка** | Долбите код, который не принимается |
| **Последствие** | Временная блокировка активации |
| **Решение** | Скрин → продавец, **не** повторять ввод |

### Prime без запаса на курсе

| | |
|---|---|
| **Ошибка** | Ровно сумма Prime на балансе |
| **Последствие** | Курс сдвинулся — покупка не проходит |
| **Решение** | Запас **20–30 рупий** сверх Prime |

## TradeON и арбитраж

### Pulse без фильтра 10 продаж/сутки

| | |
|---|---|
| **Ошибка** | Купили скин «красивый, +50%» |
| **Последствие** | Hold 7 дней → скин висит неделями |
| **Решение** | Фильтр **минимум 10 продаж/сутки** |

### Покупка наугад без графика

| | |
|---|---|
| **Ошибка** | Не сверили график за неделю |
| **Последствие** | Накрутка перед дампом, минус |
| **Решение** | График + текущий лот на маркете Steam |

### Бесплатный Pulse и дорогие скины

| | |
|---|---|
| **Ошибка** | Пытаетесь вручную найти скин > $1 |
| **Последствие** | Pulse не показывает — слепая покупка |
| **Решение** | Диапазон **от $0.20**, или платная версия |

### Комиссия и USDT после сделки

| | |
|---|---|
| **Ошибка** | Посчитали прибыль до вывода |
| **Последствие** | «Минус» вместо «+10%» |
| **Решение** | Комиссия + курс USDT **до** листинга |

## Retake, Sandbox, фарм

### Retake на 8 клиентах со слабым ПК

| | |
|---|---|
| **Ошибка** | 8 клиентов на 16 GB RAM |
| **Последствие** | Вылеты, AFK-кики, lvl не засчитан |
| **Решение** | Режьте до **5–6** клиентов |

### Avast без папки config.def

| | |
|---|---|
| **Ошибка** | Пропустили трюк с config.def |
| **Последствие** | «Продукт не поддерживается в вашем регионе» |
| **Решение** | Удалить config.def → **пустая папка** с тем же именем |

### NVIDIA драйвер не 546.65

| | |
|---|---|
| **Ошибка** | Свежий драйвер «для производительности» |
| **Последствие** | Sandbox вылетает при старте CS2 |
| **Решение** | Откат на **546.65** |

### 5v5 днём в пик онлайна

| | |
|---|---|
| **Ошибка** | Фарм 5v5 в прайм-тайм |
| **Последствие** | Репорты, длинные матчи, баны |
| **Решение** | **2v2**, ночной слот через SRT |

### SRT без учёта часового пояса

| | |
|---|---|
| **Ошибка** | Запуск в 15:00 по МСК для «индийских» серверов |
| **Последствие** | Очередь 20 мин, лишнее электричество |
| **Решение** | **01:00–12:00** по **местному** времени региона |

### Панель без обновления под патч

| | |
|---|---|
| **Ошибка** | Панель неделю не обновлялась после патча CS2 |
| **Последствие** | Неделя простоя фарма |
| **Решение** | Брать панель со **свежим** патчем |

### Панель = защита от бана

| | |
|---|---|
| **Ошибка** | «Купил FarmLabs — бан невозможен» |
| **Последствие** | Бан всё равно возможен |
| **Решение** | Панель экономит **время**, не **риск** |

### AFK > 1 минуты между Retake

| | |
|---|---|
| **Ошибка** | Переключение аккаунтов с паузами |
| **Последствие** | Кик, раунд не засчитан |
| **Решение** | Между аккаунтами — **< 1 мин** |

### Прокачка на FunPay без записи

| | |
|---|---|
| **Ошибка** | Заказали lvl за $1 без контроля |
| **Последствие** | Читы → бан через неделю |
| **Решение** | Требуйте **запись экрана** |

## Вывод и буфер

### Дроп напрямую на хранилку

| | |
|---|---|
| **Ошибка** | Ферма → хранилка без буфера |
| **Последствие** | Красная табличка, ограничения |
| **Решение** | **Ферма → буфер → хранилка** |

### Один перевод на $500

| | |
|---|---|
| **Ошибка** | Жирный трейд на хранилку |
| **Последствие** | Подозрение Valve |
| **Решение** | Несколько **мелких** в разные дни |

### Буфер без hold-дисциплины

| | |
|---|---|
| **Ошибка** | Трейд до окончания 7-дневного hold |
| **Последствие** | Ограничения, отмена |
| **Решение** | Ждать **hold** |

### Сбор на буфер чаще раза в месяц

| | |
|---|---|
| **Ошибка** | Каждую неделю все 50 ферм на один буфер |
| **Последствие** | Палевный луч в одну точку |
| **Решение** | **Раз в месяц**, 1 буфер на 5–10 ферм |

### Один маршрут вывода на 200 аккаунтов

| | |
|---|---|
| **Ошибка** | Всегда только ключи или только TradeON |
| **Последствие** | Палевный паттерн |
| **Решение** | Чередовать **раз в квартал** |

### Ключи в пик после патча

| | |
|---|---|
| **Ошибка** | Все фермеры скупают ключи одновременно |
| **Последствие** | Цена взлетела, выгода −40% → −55% |
| **Решение** | Считать **спред** каждый раз |

## Операционка и масштаб

### Нет таблицы аккаунтов

| | |
|---|---|
| **Ошибка** | «На 5 аккаунтах помню» → масштаб до 50 |
| **Последствие** | Смешали прокси, IP, Guard |
| **Решение** | Таблица **после каждого** аккаунта |

### 10 → 100 за неделю

| | |
|---|---|
| **Ошибка** | Масштаб без отладки |
| **Последствие** | Одна ошибка × 100 |
| **Решение** | **3 аккаунта** → полный цикл → потом рост |

### Личный Steam в фермерском клиенте

| | |
|---|---|
| **Ошибка** | Авторизация «с основного» в песочнице |
| **Последствие** | Связь личного и фермы |
| **Решение** | **Отдельная вселенная** для фермы |

- [ ] Gift: регион, чек, разнесение покупок

- [ ] Retake: ≤7 клиентов, Sandbox, драйвер 546.65
- [ ] SRT: ночной слот, 2v2
- [ ] Буфер: раз в месяц, hold, не на хранилку напрямую
- [ ] Вывод: калькулятор, чередование методов
